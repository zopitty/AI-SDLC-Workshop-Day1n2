## Date Calculation Examples

### Daily Recurrence
```
Input:  "2026-12-25T09:00:00+08:00"
Pattern: daily
Output: "2026-12-26T09:00:00+08:00"

Calculation: date.setDate(date.getDate() + 1)
```

### Weekly Recurrence
```
Input:  "2026-12-25T09:00:00+08:00" (Friday)
Pattern: weekly
Output: "2027-01-01T09:00:00+08:00" (Friday, +7 days)

Calculation: date.setDate(date.getDate() + 7)
```

### Monthly Recurrence (Normal)
```
Input:  "2026-01-15T10:00:00+08:00"
Pattern: monthly
Output: "2026-02-15T10:00:00+08:00"

Calculation: date.setMonth(date.getMonth() + 1)
```

### Monthly Recurrence (Edge Case: Month End)
```
Input:  "2026-01-31T10:00:00+08:00"
Pattern: monthly
Output: "2026-02-28T10:00:00+08:00"  â† Adjusted!

Reason: Feb doesn't have 31 days
JavaScript automatically adjusts to last valid day
```

### Yearly Recurrence (Edge Case: Leap Year)
```
Input:  "2024-02-29T10:00:00+08:00" (leap year)
Pattern: yearly
Output: "2025-02-28T10:00:00+08:00"  â† Adjusted!

Reason: 2025 is not a leap year
JavaScript automatically adjusts to Feb 28
```
